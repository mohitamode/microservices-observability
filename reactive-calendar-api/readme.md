# Google Calendar Event API

A streamlined interface for managing calendar events through structured JSON payloads. This project focuses on efficient data handling and API integration.

## Usage

To interact with the API, use the following curl command:

```bash
curl -X POST -H "Content-Type: application/json" -H "Accept: application/json" -H "Google-Calendar-API-User-Key: 12345" -d "{ \"item\":\"blah\" }" http://localhost:8070/api/v1/event
```
